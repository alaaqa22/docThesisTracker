Verwaltungs Hilfe

10. Fakultätliste

Die "Fakultäten" Seite ermöglicht Administratoren das Verwalten der Fakultäten in der Anwendung.

Auf dieser Seite haben Sie die Möglichkeit, neue Fakultäten hinzuzufügen, bestehende Fakultäten umzubenennen oder zu entfernen.

Um eine neue Fakultät hinzuzufügen, geben Sie den Namen in das entsprechende Feld ein und klicken Sie auf "Add Faculty". Die Tabelle wird automatisch aktualisiert, um die neue Fakultät anzuzeigen. Wenn der Name der neuen Fakultät bereits existiert, wird eine Fehlermeldung angezeigt.

In der Tabelle werden die vorhandenen Fakultäten aufgelistet. Sie können eine Fakultät entfernen, indem Sie auf den "Remove"-Button in der entsprechenden Zeile klicken. Sie können auch den Namen einer Fakultät ändern, indem Sie den Namen in das Textfeld in der entsprechenden Zeile eingeben und auf "Rename" klicken. Bitte beachten Sie, dass der Name eindeutig sein muss und keine bereits vorhandene Fakultät dupliziert werden kann.

Falls Fehler auftreten, wird eine Fehlermeldung angezeigt. Die Tabelle wird automatisch aktualisiert, um Änderungen anzuzeigen.
