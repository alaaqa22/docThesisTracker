9. Benutzerliste

Die "Benutzerliste" Seite zeigt eine Liste der registrierten Benutzer an.

Auf dieser Seite haben Sie verschiedene Filteroptionen, um die Benutzerliste nach Ihren Bedürfnissen anzupassen. Sie können entweder nach Fakultät filtern, indem Sie eine bestimmte Fakultät auswählen, oder nach Benutzerstatus filtern, um Benutzer mit einem bestimmten Status anzuzeigen.

Die Benutzerliste wird in einer Tabelle dargestellt und enthält Informationen wie Vorname, Nachname und E-Mail-Adresse jedes Benutzers. Sie können auch nach bestimmten Benutzern suchen, indem Sie die entsprechenden Felder ausfüllen. Wenn Sie auf den Namen eines Benutzers klicken, gelangen Sie zum Profil dieses Benutzers.

Falls keine Ergebnisse angezeigt werden, können Sie Ihre Filtereinstellungen ändern und es erneut versuchen.

Am unteren Ende der Seite finden Sie Paginierungskontrollen, mit denen Sie zwischen den Seiten der Benutzerliste navigieren können. Sie können auf "First" klicken, um zur ersten Seite zu gelangen, "Previous" für die vorherige Seite, "Next" für die nächste Seite und "Last" für die letzte Seite.
